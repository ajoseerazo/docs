---
title: "Get Disbursement"
api: "GET https://api.getmeru.com/external/disbursements/:disbursement-id"
description: "This endpoint gets the disbursement and the process status for each user"
---

### Params

<ParamField path="disbursement-id" type="string" required>
  The id of the disbursement to consult
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="disbursements" type="array">
  A list of disbursements statuses

  <Expandable title="Array of disbursements">
    <ResponseField name="id" type="string">
      Id of the disbursement for this user
    </ResponseField>
    <ResponseField name="email" type="string" required>
      User's email who received the money
    </ResponseField>
    <ResponseField name="amount" type="number" required>
      Amount sent
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl --location --request GET 'https://api.getmeru.com/external/disbursements/:disbursement-id' \
--header 'Content-Type: application/json' \
--header 'Authorization: Basic <token>'
```

</RequestExample>
