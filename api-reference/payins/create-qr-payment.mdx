---
title: "Create QR Code Payment"
api: "POST https://stablecoin-api.getmeru.com/api/v1/payins/qr-code"
description: "This endpoint allows merchants to create QR code payments for payment collection"
---

### Body

<ParamField body="orderId" type="string" required>
  The merchant's order identifier. This should be unique per order and used for tracking purposes.
</ParamField>

<ParamField body="amount" type="number" required>
  The total amount to be paid (excluding tax). Must be greater than 0.
</ParamField>

<ParamField body="taxAmount" type="number">
  The tax amount for the order. Must be greater than or equal to 0. This field is optional.
</ParamField>

<ParamField body="paymentMethod" type="string" required>
  The payment method to use for the QR code. Must be either "meru_qr_code" or "bo_qr_code".
</ParamField>

<ParamField body="onBehalfOf" type="string">
  The identifier of the entity on whose behalf the payment is being made. This field is optional.
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the QR code payment was created successfully.
</ResponseField>

<ResponseField name="paymentId" type="string">
  The unique payment identifier generated by our system. Use this ID to track the payment status.
</ResponseField>

<ResponseField name="qrCode" type="string">
  The QR code payment image encoded in base64 format. This can be directly embedded in HTML img tags or displayed in mobile apps.
</ResponseField>

<ResponseField name="expiresAt" type="string">
  The ISO 8601 timestamp when the QR code payment will expire. After this time, the QR code payment becomes invalid and a new one must be generated.
</ResponseField>



<RequestExample>

```bash cURL
curl --location --request POST 'https://stablecoin-api.getmeru.com/api/v1/payins/qr-code' \
--header 'Content-Type: application/json' \
--header 'api-key: <api-key>' \
--data-raw '{
    "orderId": "ORD-12345",
    "amount": 99.99,
    "taxAmount": 8.99,
    "paymentMethod": "meru_qr_code"
}'
```

```js Javascript
const response = await fetch('https://stablecoin-api.getmeru.com/api/v1/payins/qr-code', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Basic <token>'
  },
  body: JSON.stringify({
    orderId: 'ORD-12345',
    amount: 99.99,
    taxAmount: 8.99,
    paymentMethod: 'meru_qr_code'
  })
});

const data = await response.json();
```

</RequestExample>
