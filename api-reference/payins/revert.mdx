---
title: "Refund Payin"
api: "POST https://stablecoin-api.sandbox.getmeru.com/v1/payins/:id/revert"
description: "Refunds a payin (only allowed when the payin is already funds_received or payment_processed)."
---

### Params

<ParamField path="id" type="string" required>
  Payin internal id.
</ParamField>

### Body

<ParamField body="reason" type="string">
  Refund reason.
</ParamField>

### Response

<ResponseExample>

```json 200
{ "success": true, "message": "Payin refunded successfully" }
```

</ResponseExample>

### Error Responses

<ResponseExample>

```json 404
{ "success": false, "error": "Not Found" }
```

```json 409
{ "success": false, "error": "Conflict" }
```

</ResponseExample>
