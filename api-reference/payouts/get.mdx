---
title: "Get Payout"
api: "GET https://stablecoin-api.getmeru.com/api/v1/payouts/:payout-id"
description: "This endpoint gets the payout and the process status for each user"
---

### Params

<ParamField path="payout-id" type="string" required>
  The id of the payout to consult
</ParamField>

### Response

<ResponseField name="success" type="boolean">
  Indicates whether the call was successful.
</ResponseField>

<ResponseField name="payouts" type="array">
  A list of payouts statuses

  <Expandable title="Array of payouts">
    <ResponseField name="id" type="string">
      Id of the payout for this user
    </ResponseField>
    <ResponseField name="email" type="string" required>
      User's email who received the money
    </ResponseField>
    <ResponseField name="amount" type="number" required>
      Amount sent
    </ResponseField>
    <ResponseField name="status" type="string" required>
      Status of the payout (COMPLETED, FAILED, PENDING)
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl --location --request GET 'https://stablecoin-api.getmeru.com/api/v1/payouts/:payout-id' \
--header 'Content-Type: application/json' \
--header 'api-key: <api-key>'
```

</RequestExample>
