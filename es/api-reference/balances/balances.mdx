---
title: "Obtener saldos de la empresa"
api: "GET https://stablecoin-api.sandbox.getmeru.com/v1/companies/balances"
description: "Este endpoint obtiene el saldo actual de una empresa en todas las monedas soportadas"
---

### Respuesta

<ResponseField name="success" type="boolean">
  Indica si la operación fue exitosa.
</ResponseField>

<ResponseField name="companyId" type="string">
  El identificador de la empresa para la cual se devuelven los saldos.
</ResponseField>

<ResponseField name="balances" type="object">
  Objeto con los saldos de cada moneda soportada
  <Expandable title="Objeto balances">
    <ResponseField name="bob" type="number">
      Saldo en boliviano (BOB)
    </ResponseField>
    <ResponseField name="ars" type="number">
      Saldo en peso argentino (ARS)
    </ResponseField>
    <ResponseField name="brl" type="number">
      Saldo en real brasileño (BRL)
    </ResponseField>
    <ResponseField name="usdc" type="number">
      Saldo en USD Coin (USDC)
    </ResponseField>
    <ResponseField name="usdt" type="number">
      Saldo en Tether (USDT)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="lastUpdated" type="string">
  La fecha y hora (ISO 8601) de la última actualización de los saldos.
</ResponseField>

<RequestExample>

```bash cURL
curl --location --request GET 'https://stablecoin-api.sandbox.getmeru.com/v1/companies/balances' \
--header 'Content-Type: application/json' \
--header 'api-key: <api-key>'
```

```js Javascript
const response = await fetch(
  "https://stablecoin-api.sandbox.getmeru.com/v1/companies/balances",
  {
    method: "GET",
    headers: {
      "Content-Type": "application/json",
      "api-key": "<api-key>",
    },
  }
);

const data = await response.json();
```

</RequestExample>
