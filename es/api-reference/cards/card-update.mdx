---
title: "Actualizar tarjeta"
api: "PATCH https://stablecoin-api.sandbox.getmeru.com/api/v1/cards-program/cards/:cardId"
description: "Este endpoint actualiza la configuración de la tarjeta, incluyendo estado, límites y ajustes"
---

### Parámetros

<ParamField path="cardId" type="string" required>
  El identificador único de la tarjeta
</ParamField>

### Cuerpo

<ParamField body="status" type="string">
  Estado de la tarjeta: `notActivated`, `active`, `locked` o `canceled`
</ParamField>

<ParamField body="limit" type="object">
  Configuración del límite de gasto

  <Expandable title="Objeto limit">
    <ParamField body="amount" type="number" required>
      Monto del límite
    </ParamField>
    <ParamField body="frequency" type="string" required>
      Frecuencia: `per24HourPeriod`, `per7DayPeriod`, `per30DayPeriod`, `perYearPeriod`, `allTime` o `perAuthorization`
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="configuration" type="object">
  Ajustes de configuración de la tarjeta

  <Expandable title="Objeto configuration">
    <ParamField body="virtualCardArt" type="string" required>
      Identificador del arte de la tarjeta virtual
    </ParamField>
  </Expandable>
</ParamField>

### Respuesta

<ResponseField name="success" type="boolean">
  Indica si la operación fue exitosa
</ResponseField>

<ResponseField name="cardId" type="string">
  El identificador único de la tarjeta
</ResponseField>

<ResponseField name="status" type="string">
  Estado actualizado de la tarjeta
</ResponseField>

<RequestExample>

```bash cURL
curl --location --request PATCH 'https://stablecoin-api.sandbox.getmeru.com/api/v1/cards-program/cards/7a8b9c0d-1e2f-3a4b-5c6d-7e8f9a0b1c2d' \
--header 'Content-Type: application/json' \
--header 'api-key: <api-key>' \
--data-raw '{
    "limit": {
        "amount": 10000,
        "frequency": "per30DayPeriod"
    }
}'
```

```js Javascript
const response = await fetch('https://stablecoin-api.sandbox.getmeru.com/api/v1/cards-program/cards/7a8b9c0d-1e2f-3a4b-5c6d-7e8f9a0b1c2d', {
  method: 'PATCH',
  headers: {
    'Content-Type': 'application/json',
    'api-key': '<api-key>'
  },
  body: JSON.stringify({
    limit: {
      amount: 10000,
      frequency: "per30DayPeriod"
    }
  })
});

const data = await response.json();
```

</RequestExample>

<ResponseExample>

```json 200
{
  "success": true,
  "cardId": "7a8b9c0d-1e2f-3a4b-5c6d-7e8f9a0b1c2d",
  "status": "active"
}
```

</ResponseExample>
