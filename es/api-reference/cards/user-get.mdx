---
title: "Obtener detalles del usuario"
api: "GET https://stablecoin-api.sandbox.getmeru.com/api/v1/cards-program/users/:userId"
description: "Este endpoint obtiene información detallada de un usuario"
---

### Parámetros

<ParamField path="userId" type="string" required>
  El identificador único del usuario
</ParamField>

### Respuesta

<ResponseField name="success" type="boolean">
  Indica si la operación fue exitosa
</ResponseField>

<ResponseField name="userId" type="string">
  El identificador único del usuario
</ResponseField>

<ResponseField name="firstName" type="string">
  Nombre del usuario
</ResponseField>

<ResponseField name="lastName" type="string">
  Apellido del usuario
</ResponseField>

<ResponseField name="email" type="string">
  Correo electrónico del usuario
</ResponseField>

<ResponseField name="status" type="string">
  Estado de la cuenta del usuario: `pending`, `active`, `suspended` o `closed`
</ResponseField>

<ResponseField name="phoneNumber" type="string">
  Número de teléfono del usuario con código de país
</ResponseField>

<ResponseField name="walletAddress" type="string">
  Dirección de wallet blockchain asociada
</ResponseField>

<ResponseField name="createdAt" type="string">
  Fecha y hora (ISO 8601) de cuándo se creó la cuenta del usuario
</ResponseField>

<ResponseField name="updatedAt" type="string">
  Fecha y hora (ISO 8601) de la última actualización de la cuenta
</ResponseField>

<RequestExample>

```bash cURL
curl --location --request GET 'https://stablecoin-api.sandbox.getmeru.com/api/v1/cards-program/users/3ddd0e5b-6276-4b48-b756-c1fcc9a2efd1' \
--header 'Content-Type: application/json' \
--header 'api-key: <api-key>'
```

```js Javascript
const response = await fetch('https://stablecoin-api.sandbox.getmeru.com/api/v1/cards-program/users/3ddd0e5b-6276-4b48-b756-c1fcc9a2efd1', {
  method: 'GET',
  headers: {
    'Content-Type': 'application/json',
    'api-key': '<api-key>'
  }
});

const data = await response.json();
```

</RequestExample>
