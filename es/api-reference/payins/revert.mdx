---
title: "Reembolsar payin"
api: "POST https://stablecoin-api.sandbox.getmeru.com/v1/payins/:id/revert"
description: "Reembolsa un payin (solo permitido cuando el payin ya está en funds_received o payment_processed)."
---

### Parámetros

<ParamField path="id" type="string" required>
  Id interno del payin.
</ParamField>

### Cuerpo

<ParamField body="reason" type="string">
  Motivo del reembolso.
</ParamField>

### Respuesta

<ResponseExample>

```json 200
{ "success": true, "message": "Payin refunded successfully" }
```

</ResponseExample>

### Respuestas de error

<ResponseExample>

```json 404
{ "success": false, "error": "Not Found" }
```

```json 409
{ "success": false, "error": "Conflict" }
```

</ResponseExample>
