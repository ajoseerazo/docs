---
title: "Listar payouts"
api: "GET https://stablecoin-api.sandbox.getmeru.com/v1/payouts"
description: "Devuelve los payouts en una lista dentro de data, con metadatos de paginación."
---

### Parámetros de consulta

<ParamField query="page" type="number">
  Número de página (por defecto 1).
</ParamField>

<ParamField query="pageSize" type="number">
  Tamaño de página (por defecto 20, máximo 100).
</ParamField>

<ParamField query="startDate" type="string">
  Cota inferior inclusiva sobre `createdAt` (ISO 8601). Opcional.
</ParamField>

<ParamField query="endDate" type="string">
  Cota superior inclusiva sobre `createdAt` (ISO 8601). Opcional.
</ParamField>

<Note>
  `endDate` es un timestamp, no un día calendario: `2024-01-31` equivale a `2024-01-31T00:00:00Z` y excluye ese día — para incluir el día completo, envía `2024-01-31T23:59:59Z`. Cuando se envían ambas fechas, `startDate` debe ser anterior o igual a `endDate`; de lo contrario la API responde `400 Bad Request`.
</Note>
